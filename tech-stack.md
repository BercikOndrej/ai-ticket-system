## Tech Stack

### Frontend

- **React** (Vite)
- **React Router** for navigation

### Backend

- **TypeScript**
- **Express.js**

### Database

- **PostgreSQL** + **Prisma** ORM

### Authentication

- Database-managed sessions

### AI

- **Claude API** (Anthropic)

### Email ingestion

- **SendGrid** or **Mailgun** webhooks

### Deployment

- **Docker** + **cloud provider** (AWS, Railway, DigitalOcean)
